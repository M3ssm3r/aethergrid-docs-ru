# Руководство пользователя

## Обзор интерфейса

Экран разделен на две основные части:

1.  **Основная область (Карта сети):** Динамический граф, который показывает узлы сети и связи между ними.
2.  **Боковая панель (Панель анализа):** Отображает подробную информацию о выбранном узле и ведет лог событий.

## Как читать карту

*   **Узлы (кружки):** Это компоненты сети (серверы, фаерволы). Цвет показывает их статус:
    *   **Голубой:** Нормальное состояние.
    *   **Красный:** Узел скомпрометирован или является источником атаки.
*   **Связи (линии):** Показывают соединения.
    *   **Серые:** Обычный трафик.
    *   **Красные (с движущимися частицами):** Активная атака по этому каналу.

## Порядок работы

1.  Наблюдайте за картой, ищите красные линии или узлы.
2.  Кликните на подозрительный узел, чтобы справа загрузилась его детальная информация.
3.  Сопоставьте информацию об узле с логом событий, чтобы понять цепочку атаки.

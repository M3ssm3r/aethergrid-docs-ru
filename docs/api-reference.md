# Справка по API

## GET /api/v1/nodes/{node_id}

Получает детальный статус конкретного узла сети в реальном времени.

### Параметры пути

| Параметр | Тип    | Описание                   |
|----------|--------|----------------------------|
| `node_id`| string | **Обязательно.** ID нужного узла. |

### Пример ответа (200 OK)

```json
{
  "id": "web-server-01",
  "ip": "192.168.1.10",
  "type": "Nginx на Ubuntu 22.04",
  "status": "СКОМПРОМЕТИРОВАН",
  "cpu_load": 0.92,
  "memory_usage": 0.78
}
```

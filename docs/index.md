# Введение в AetherGrid

AetherGrid: Threat Visualizer - это интерфейс-симулятор для мониторинга сетевых атак в реальном времени. Он показывает карту инфраструктуры и визуализирует, как векторы атаки проникают в систему.

Продукт предназначен для аналитиков SOC (центров кибербезопасности), чтобы они могли быстро получить представление о ситуации во время взлома.
